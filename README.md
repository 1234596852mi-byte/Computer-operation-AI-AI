This program enables AI to perform human-like tasks on Windows using natural language commands.

The initial goal of creating this program was to allow AI to work fully automatically, freeing up your hands.

I am a C# programmer and do not know Python; this code is entirely written in Gemini-ai.

First, there is an AI chat log, including operation steps, which shows the AI's logic. When the AI ​​sends multiple commands to Windows, it will encounter an error during execution, and the execution will terminate.

So please use it with confidence.

# Function Introduction

1. Left/Right Mouse Click

2. Double Mouse Click

3. Keyboard Operation, which can simulate pressing multiple keys, such as "Win" + "D"

4. !!!!!! When text recognition is required, an image server must be enabled!!!!!!

5. !!!!!! The AI ​​path has been saved in E:\Jiang\Test\AI Log. Please set it in the code to change it yourself.

6. !!!!!! The image server path may vary; check the image server's IP address.!!!!!! 

7. !!!!!! When sending information to AI, add "abc" to the end to send a screenshot of your current desktop.!!!!!! 

8. !!!!!! When opening the file, download the contents of "aisummary.txt" sent by AI; otherwise, AA will not send the Jason command.!!!!!!

9. English speakers can use a translation tool to translate the contents of "ai总结.txt" into English and send it to the AI.
